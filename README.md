# websocket-simple-chat-server
An example of a chat server using websocket

## Application

### Requirements

java>=8.0
maven (mvn)

### Install

```commandline
$ mvn clean
$ mvn compile
$ mvn install
```

or use the shortcut

```commandline
$ bash install.sh
```

### How to use

Run the application with

```commandline
$ java -jar target chatserver-1.0-dev-jar-with-dependencies.jar
```

Chatserver is now listen to 0.0.0.0:9001
